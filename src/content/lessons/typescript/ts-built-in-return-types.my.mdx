> TypeScript က ကိုယ့် function တွေအသက်သာမက၊ JavaScript built-in function တွေရဲ့ return type ကိုပါ စစ်ပါတယ်။

### TypeScript က built-in return types ကို သိတယ်

- **အဓိက takeaway:** TypeScript က ကိုယ့် code ပဲမဟုတ်၊ built-in JS function တွေရဲ့ type ကိုပါ အတိအကျ enforce လုပ်တယ်။
- ဥပမာ:
    - `prompt()` → `string | null`
    - `confirm()` → `boolean`
    - `document.getElementById()` → `HTMLElement | null`
- ဒါကြောင့် built-in API သုံးတဲ့အခါ type mismatch ကို compile-time မှာပဲ ဖမ်းလို့ရတယ်။

Good Practice:
- user input ကလို အပြင် sources တွေက type ကို အမြဲစစ်ပါ။
- `typeof`, `instanceof` နဲ့ narrow လုပ်ပါ။
- `Number()`, `String()` စတာနဲ့ type ပြောင်းတဲ့အခါ invalid case ကို handle ပေးပါ။
- TypeScript ရဲ့ knowledge ကို ရင်းနှီးဖို့—it prevents runtime errors.
